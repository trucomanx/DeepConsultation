# deep_consultation

Deep consultation Learning Tools


## Install from PYPI

The homepage in pipy is https://pypi.org/project/deep_consultation/

```bash
pip install --upgrade deep_consultation
```

## Install from source

Installing `deep_consultation` 

```bash
git clone https://github.com/trucomanx/DeepConsultation.git
cd DeepConsultation
pip install -r requirements.txt
cd src
python -m build
pip install dist/deep_consultation-*.tar.gz
```

## Uninstall

```bash
pip uninstall deep_consultation
```
